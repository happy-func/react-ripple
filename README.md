# react-ripple

## Getting Started

Install dependencies,

```bash
# npm
$ npm i react-ripple
# yarn
$ yarn add react-ripple
```

## Usage

### Basic

```javascript
import React from 'react';
import ReactRippleButton from 'react-ripple';
import styles from './basic.module.less';

export default () => {
  return (
    <ReactRippleButton>
      <div className={styles.button}>React Ripple Button</div>
    </ReactRippleButton>
  );
};
```
